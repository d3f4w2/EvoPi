# 灵感池（捕获层）

> 任何新输入先记这里——**10 秒一行，不评判、不打断手头的事**。评判留到有整块时间时集中做（见 演化流程.md 第②③步）。
>
> 状态流转：`🆕 待提炼` → `🔬 已提炼待定性` → 定性为 `A 新功能` / `B 架构优化` / `C 推翻旧决策` / `D 不用`。
> 升级到 路线图 或写了 ADR 后，在这里标 `→ 已升级(链接)`，条目留档不删（记录你思考过什么）。

---

## 条目模板（复制这段往下加）

```markdown
### YYYY-MM-DD · <一句话标题>
- **来源**：文章链接 / repo / 自己的灵感 / "用的时候发现___"
- **它主张什么**：（提炼时补，两三句）
- **对本项目的可借鉴点**：（哪个部分？加什么/改什么）
- **定性**：🆕待提炼 →（A新功能 / B架构优化 / C推翻旧决策 / D不用）
- **去向**：留池 / → 路线图 / → ADR-000X / ✗不用（理由：___）
```
